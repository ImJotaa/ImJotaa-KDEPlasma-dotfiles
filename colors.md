# Paleta — "Neon Grid" (retro cyberpunk, minimalista)

Referencia única de color para todos los dotfiles. Si cambias algo aquí, propágalo.

## Base

| Rol | Hex | Uso |
| --- | --- | --- |
| bg0 | `#0d0221` | Fondo principal (negro violáceo) |
| bg1 | `#1a1333` | Fondo alternativo, barras, popups |
| bg2 | `#241b3e` | Superficies elevadas, selección |
| fg | `#dee4ff` | Texto principal |
| muted | `#6f6a8f` | Texto secundario, bordes inactivos |
